Module 1: Setup
Consent. Student reads and submits the consent form. Required to proceed.
Pre-survey. Baseline questionnaire. Submitting it triggers automatic group randomization on the backend (VOICE_FIRST vs SIMCASE_FIRST), but the group is not revealed yet.
Module 2: Phase 1
Phase 1 tutorial. Reveals the student's assigned group and explains what's coming.
Phase 1A activity. Either an assignment or SimuCase (a page of instructions + external link + recording upload), depending on the group.
Phase 1A survey. Post-condition survey.
Phase 1B activity. The opposite modality from 1A, every student experiences both.
Phase 1B survey. Same question set as 1A survey, different instance.

Module 4: Phase 3
Assignment: AI feedback is auto-generated but not revealed to the student. Two professor reviewers each write their own feedback independently. All three feedbacks stored as blinded (no author shown to the student).
Survey: Locked until both reviewers have submitted. Submitting the survey reveals the true sources.

🔍 Scene Dropdown Debug Helper

Instructions:

  1. Open your app at http://localhost:5173/faculty/assignments/new
  2. Open the browser DevTools (F12)
  3. Go to the Console tab
  4. Copy and paste the script below into the console
  5. Press Enter to run it

📋 Debug Script (Copy this to console)

// Scene Dropdown Debug Script
console.log('🔍 Starting Scene Dropdown Diagnostic...\n');

// 1. Check if the page loaded properly
console.log('1️⃣ Checking page elements...');
const sceneDropdown = document.querySelector('[data-testid*="scene"], input[placeholder*="Select a simulation"], input[placeholder*="scene"]');
const allSelects = document.querySelectorAll('select, input[role="combobox"], [class*="Select"]');
console.log('Scene dropdown element:', sceneDropdown);
console.log('All select/combobox elements found:', allSelects.length);
allSelects.forEach((el, i) => {
  console.log(`  Select ${i}:`, el.getAttribute('placeholder') || el.textContent?.substring(0, 50));
});

// 2. Check React component state (if accessible)
console.log('\n2️⃣ Checking React component state...');
try {
  const rootElement = document.getElementById('root');
  if (rootElement && rootElement._reactRootContainer) {
    console.log('React root found');
  }
} catch (e) {
  console.log('Unable to access React internals (expected)');
}

// 3. Check for network requests
console.log('\n3️⃣ Checking Network Activity...');
console.log('Open the Network tab to see if the /scene-catalog API call was made');
console.log('Look for: GET request to endpoint containing "scene-catalog"');

// 4. Look for errors in console
console.log('\n4️⃣ Checking for errors...');
const errors = window.console.error;
console.log('Console errors should appear above (if any)');

// 5. Manual check instructions
console.log('\n5️⃣ Manual Checks:');
console.log('✓ Is the Scene dropdown visible on the page?');
console.log('✓ Can you click on the Scene dropdown?');
console.log('✓ When you click it, does a menu appear?');
console.log('✓ Is the dropdown menu empty or does it have items?');
console.log('✓ Check Network tab: Did the scene-catalog API call succeed (200)?');
console.log('✓ If API call succeeded, what was the response body?');

console.log('\n📊 Quick Visual Test:');
console.log('Run this to highlight the Scene dropdown:');
console.log('document.querySelector(\'[placeholder*="Select a simulation"]\')?.style.border = "3px solid red"');

console.log('\n✅ Diagnostic complete!');

🌐 Network Tab Checklist

  1. Look for a request to scene-catalog
  2. Check the status code (should be 200)
  3. Click on the request to see the response body
  4. The response should have format: { "scenes": [...] }

🎯 What to Report Back

  • Dropdown State: Empty or has items?
  • API Call: Success (200) or error?
  • Response Data: What did the API return?
  • Console Errors: Any red error messages?
  • Clickable: Can you click and open the dropdown?
